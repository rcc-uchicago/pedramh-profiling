The native PlaSim postprocessor is called burn7.cpp. You need also this file with the postprocessor2.0 (our). There is compiled version of this postprocessorfor each of the cluster we used so far (eg Stampede3, derecho, jeanzay, etc.). If you want to use the postprocessor for a new cluster, you will have to compile it on this new machine.
